# Summary 
This is a basic demo test project to show usage of Playwright and TypeScript.

The tests included in the project are API tests for [OpenWeather API](https://openweathermap.org/).

# Running tests
- Run tests `npx playwright test`
- Show report: `npx playwright show-report`

# Report
Reporting is provided by default via Playwright.
